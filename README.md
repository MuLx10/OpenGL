# OpenGL
Repository containing OpenGL examples implementation.

## Deps
- CMake >= 3
- g++ (MinGW in Windows)

## Build

```bash
$ mkdir build
$ cd build
$ cmake ..
$ make
```

#### Individual build
```bash
$ make {executable} # eg. make triangle
```


## Run

```bash
$ build/bin/{executable} # eg. build/bin/triangle
```
